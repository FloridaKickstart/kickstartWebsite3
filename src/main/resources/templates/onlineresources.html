Online Resources

IntelliJ

Intellij is an indexed source code IDE that allows user to easily create, run and test programs on Java with the use of intellisense and access to multiple java packages and libraries.

BlueJ

BlueJ is a quick and easy application that allow newcomers to Java to easily create projects and programs using simple formatting techniques and compiling operators.

Microsoft Visual Studio Code

Microsoft Visual Studio is an extremely useful IDE that allows users to program using multiple programming languages in the form of downloadable extensions, such as HTML, Java, Python, and Javascript. Its complex intellisense and quick navigation bar encorages programs to be created by programmers still learning different languages.

PyCharm

Pycharm is a quick and easy IDE for newcomers to Python 3 to play around and test with basic programs. It also offers a wide variety of libaries and applications for students to incorporate useful techniques or extensions on their code for more efficient use.

Brackets.io

Brackets.io is a local text editor that allows user to create websites and test them in real time, all while performing live updates on elements throughout the design process.

Atom.io

Atom.io is a simple online text editor used to create website using HTML, CSS, Javascript and JQuery in order to create animations and personalize layouts for online platforms.
